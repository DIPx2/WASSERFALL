Справочник кодов системы.

Для автоматизации обработки ошибок в Wasserfall введена внутренняя кодировка:

Код,Категория,Описание
pg_0,Успех,Команда выполнена успешно.
pg_10,Окружение,psql не найден на удаленном хосте.
pg_12,Доступ,Ошибка аутентификации в базе (Password/User).
pg_14,SQL,Синтаксическая ошибка в отрендеренном запросе.
ssh_20,Сеть,Ошибка подключения по SSH или таймаут.
ssh_99,Критическая,Непредвиденный сбой транспорта.
